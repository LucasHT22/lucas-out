# Lucas Out
An basic implementation of the classic Lights Out puzzle in Godot, with guarantee-solvable puzzle generation

## How to play

 - Click a cell to toggle it and its neighbors (up, down, left, right)
 - Solve: computes and applies a solution to board
 - Restart: generates a new solvable puzzle

## The Math

Lghts Out looks like a simple toggle game, but it's really linear algebra. Treat the board as a vector over GF(2), the field is just 2 elements, 0 and 1, where addition is XOR, each cell is a coordinate.

A click on a cell `j` is itself a vector: 1s at `j` and its neighbors, 0s everything else. Toggling is XOR, so applying a sequence of clicks just sums their vectors. Two things fallout of that immediately, and that's the reason this puzzle is solvable at all:

 - Order doesn't matter. XOR cares only what order you add things in
 - Every click undoes itself. Click a cell twice and nothing happened


That means solving the puzzle reduces to one question: which set of cells, clicked once each, sums (via XOR) to the current board state? Written as a linear system:
 
```
A · x = b   (mod 2)
```

`b` is the board you're trying to clear, `x` is the unknown set of clicks, and `A` is the N^2xN^2 matrix where column `j` is the toggle pattern for cell `j`. Solve for `x` and you have your answer!

The solver uses Gaussian elimination, the standard technique for solving linear systems, just done with XOR instaed of regular subtraction. It goes column by column, picks a row with a 1 as the pivot, and XORs that row into every other row that also has 1 in the same column. Once that's done for every column, `x` can be read off directly.

One nice guarantee: puzzles are always generated by clicking from the all-off board, so whatever pattern comes out was reached by clicks in the first place, that means it's always solvable.

## Winning Strategy

There's not really one, the closest thing to a manual is "chase it row by row": for each row from the top, click every cell directly below any light still on, pushing everything down toward the bottom row.
